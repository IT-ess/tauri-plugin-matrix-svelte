## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-ping`
- `allow-login-and-create-new-session`
- `allow-submit-async-request`
- `allow-fetch-media`
- `allow-fetch-user-profile`
- `allow-watch-notifications`
